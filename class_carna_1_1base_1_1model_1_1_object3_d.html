Detailed Description

Represents an object in 3D space.

The way Object3D instances are rendered is described here in detail.

Instances of this class are part of the data model and therefore composed by Scene objects. At the highest abstraction level, which this class describes, every such object has the only attribute position.

See also:
Provided implementations are view::Point3D, view::Polyline3DVertex and the abstract model::RotatableObject3D class.

Creating custom implementation

In order to create a custom implementation you need to define two methods:

  • paint performs all necessary GL drawing calls to render the object.
  • paintFalseColor renders the same geometry but in the specified color.

The latter one is used for a simple picking implementation. Therefore it is not just legitimate, but convenient to paint the geometry in reduced quality in paintFalseColor.

Deriving from the abstract Object3D class is as simple as:

 class MyObject3D : public Carna::base::model::Object3D
 {

 public:
 
     MyObject3D( Carna::base::model::Scene& model ) : Carna::base::model::Object3D( model ){ }


     virtual void paint( const Carna::base::view::Renderer& ) const override;

     virtual void paintFalseColor
         ( const Carna::base::view::Renderer&
         , const Carna::base::Vector3ui& ) const override;

 };

An implementation could look as follows. The code below produces a yellow dot where the object is located:

 using Carna::base::view::Renderer;
 using Carna::base::Vector3ui;
 
 void MyObject3D::paintFalseColor( const Renderer& renderer, const Vector3ui& color ) const
 {
     glColor3ub( color.x, color.y, color.z );
     glBegin( GL_POINTS );
     glVertex3f
         ( position().toVolumeUnits()[ 0 ]
         , position().toVolumeUnits()[ 1 ]
         , position().toVolumeUnits()[ 2 ] );
     glEnd();
 }

 void MyObject3D::paint( const Renderer& renderer ) const
 {
     paintFalseColor( renderer, Vector3ui( 255, 255, 0 ) );
 }
