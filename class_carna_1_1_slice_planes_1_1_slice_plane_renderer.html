Detailed Description

Abstract Renderer specialization that is supposed to utilize the SlicePlane class for rendering slice planes. This is a generalization of the DefaultSlicePlaneRenderer class.

See also:
The SlicePlaneControllerUI class provides an easy-to-use user interface for the purpose of configuring this renderer.

Specifies slice plane.

The plane is defined by \( \vec x : \langle \vec x, \frac{\vec n}{\| \vec n \|} \rangle = d\), where \( \vec n \) is normal and \( d \) is distance.

Precondition:
Carna::base::isEqual( normal.norm(), 0. ) == false
Parameters:
normalis normalized plane normal vector \( \in \mathbb{R}^3 \) in volume space coordinates.
distanceof plane to origin \( \in [0,1] \) in volume space coordinates.
borderspecifies width of inner border to draw.
Returns:
true if computation was successful.
