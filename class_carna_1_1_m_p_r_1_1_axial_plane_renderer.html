Detailed Description

Abstract base::view::Renderer specialization that is supposed to display the orthogonal intersections \( \big\{ \vec x : \langle \vec e_i, \vec x \rangle = d_i, i \in \{ 1, 2, 3 \} \big\} \) of the volume data set, where \( d_i \) is adjustable. This is a generalization of the DefaultAxialPlaneRenderer class.

See also:
This renderer is part of the AxialPlaneVisualization module.

The interface expects the rendered perspective to be fully determined by the utilized camera. This means in particular that the camera determines which planes actually are visible.

The intersection point \( ( d_1, d_2, d_3 )^\mathrm T \) defines the position of the planes. It must be set using volume space coordinates. The intersection point is the one point that is common to all three cross-sectional planes.

Customizations

Disable colored plane indicators
 static void disableSliderPainter( Carna::MPR::AxialPlaneRenderer& renderer )
 {
     renderer.setSliderPainter( new Carna::MPR::NullSliderPainter() );
 }

Also disable the picking ability, else the user might click some plane sliders that actually are invisible:

Disable ability to pick plane slider
 static void disablePicking( Carna::MPR::AxialPlaneRenderer& renderer )
 {
     renderer.setSliderScreenSpaceBuffer( new Carna::MPR::NullSliderScreenSpaceBuffer() );
 }
