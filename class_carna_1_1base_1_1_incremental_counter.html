Detailed Description

Provides cyclic incremental counter that can be controlled by signals and slots.

Since:
API Version 2.1 Release 4

The counter is initialized with 0 and any maximum value threshold \(\geq 1\). The value is incremented by invoking increment. As soon as the value reaches the preset threshold, it is reset to 0 and the thresholdReached signal is emitted.
