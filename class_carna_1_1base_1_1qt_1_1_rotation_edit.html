Detailed Description

Manipulating rotations in 3D space.

User interaction and behavior

Pan and tilt can be adjusted by holding the primary mouse button pressed upon the view while moving the cursor around.

The scroll wheel can be used to adjust the spin quickly, when not hovering any input field with the cursor; the hovered attribute (pan, tilt or spin) is manipulated else.

Details

The rotation is described by three degrees of freedom:

  • rotation around own x-axis, which is called spin
  • rotation around parent z-axis, which is tilting the parent xy-plane
  • rotation around tilted y-axis, which is rotation in parent xy-plane (pan)

The spin is not visualized by the view.
