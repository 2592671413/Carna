Detailed Description

template<typename KeyType, typename CountingType = unsigned int, typename ConstReferencedKeyType = const KeyType&>
struct Carna::base::model::IndexedCounter< KeyType, CountingType, ConstReferencedKeyType >

Counter specialization that uses std::vector as the underlying data container.

This specialization can only be used if KeyType can be considered an index, e.g. either it is an integral data type or can it be converted to such implicitly. Furthermore the usage of this specialization is to be preferred over AssociativeCounter whenever the counted keys are luckily to leave no gaps within KeyType.

Parameters:
KeyTypeis the object type whose occurrences are to be counted.
ContingTypedefines the counting domain.
ConstReferencedKeyTypedefines the type to be used for referencing objects of type KeyType constantly. Depending on the size of KeyType it can be more efficient to use a copy instead of a reference by setting this type argument to KeyType.
