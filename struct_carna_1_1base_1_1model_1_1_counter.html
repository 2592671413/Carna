Detailed Description

template<typename Container, typename KeyType, typename CountingType = unsigned int, typename ConstReferencedKeyType = const KeyType&>
struct Carna::base::model::Counter< Container, KeyType, CountingType, ConstReferencedKeyType >

Class template for generalized modeling of \(\text{KeyType}\to\text{CountingType}\) mappings with \(\text{CountingType} \subset \mathbb{N}_0\).

The standard library types std::vector and std::map are commonly used for Container. Refer to the specializations IndexedCounter and AssociativeCounter of this class template. Hints on when to use which of these containers are given there.

Parameters:
Containermust support element access through the index operator [const KeyType&]. Besides that, it must provide a method clear that removes all elements.
KeyTypeis the object type whose occurrences are to be counted.
ContingTypedefines the counting domain.
ConstReferencedKeyTypedefines the type to be used for referencing objects of type KeyType constantly. Depending on the size of KeyType it can be more efficient to use a copy instead of a reference by setting this type argument to KeyType.
