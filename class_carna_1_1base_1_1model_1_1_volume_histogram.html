Detailed Description

template<typename CounterType = IndexedCounter< signed short, unsigned int, signed short >>
class Carna::base::model::VolumeHistogram< CounterType >

Computes the histogram of Volume objects.

Usually it is sufficient to use either IndexedCounter as the underlying

\[\mathrm{FieldCodomain}(\text{ScalarFieldType})\to\text{CountingType}\]

realization. Refer to the particular counter's documentation for details.

Parameters:
ConcreteCounterdefines the \(\mathrm{FieldCodomain}(\text{ScalarFieldType})\to\text{CountingType}\) realization.

Example with volume as an Volume typed object reference: 

 Carna::base::model::VolumeHistogram histogram( volume, volume.size );
 histogram.updateCounter();
 for( signed short huv = -1024; huv <= 3071; ++huv )
 {
     const unsigned int absolute_frequency = histogram.counter[ huv ];
     report_histogram_sample( huv, absolute_frequency );
 }
See also:
The class template HistogramClassDetector provides an algorithm for automatic detection of histogram classes.
