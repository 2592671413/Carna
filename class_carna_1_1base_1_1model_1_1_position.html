Detailed Description

Defines a position in model- or volume space.

Attention:
Priorly to 1.9, volume space and volume units were called model space and model units, whereas model unit is now an equivalent term for millimeters since 1.9.

static Transformation Carna::base::model::Position::toMillimetersTransformation ( const Scene &  ) [static]

Returns the matrix of the toVolumeUnits operation.

The returned matrix is \(M_0\) from:

\[ v_\text{modelspace} = M_0 \cdot v_\text{volumespace} \]

Returns the matrix of the toMillimeters operation.
