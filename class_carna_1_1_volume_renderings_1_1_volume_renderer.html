Detailed Description

Abstract Renderer specialization that is supposed to utilize Ray Marching as it's rendering technique. This is a generalization of the DefaultVolumeRenderer class.

When the ray marching is finished, a further rendering pass is performed in order to apply a set of optional post processing steps. These steps include:

  • inverting colors
  • overlaying a black circular mask
  • rotating the image
  • flipping the image horizontally and/or vertically

The VolumeControllerUI class offers an easy-to-use user interface with the purpose of configuring this renderer.
