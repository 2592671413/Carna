Detailed Description

template<typename ConcreteCounter, typename ScalarFieldType, unsigned int scalar_field_dimension>
class Carna::base::model::Histogram< ConcreteCounter, ScalarFieldType, scalar_field_dimension >

Computes the histogram of any given scalar field in any given dimension.

Usually it is sufficient to use either IndexedCounter or AssociativeCounter for the underlying

\[\mathrm{FieldCodomain}(\text{ScalarFieldType})\to\text{CountingType}\]

realization. Refer to the particular counter's documentation for details.

Parameters:
ConcreteCounterdefines the \(\mathrm{FieldCodomain}(\text{ScalarFieldType})\to\text{CountingType}\) realization.
ScalarFieldTypemust define the parenthesis operator to map from \(\mathbb{N}_0^N\) with scalar_field_dimension as \(N\) to codomain values.
scalar_field_dimensiondefines the dimension of the scalar field's domain.
See also:
The class VolumeHistogram is a specialization that simplifies histogram computation for Volume objects.
The class template HistogramClassDetector provides an algorithm for automatic detection of histogram classes.
