Detailed Description

Renderer adapter for the SlicePlane class.

The render visualizes any custom slice planes. It is part of the SlicePlaneVisualization module.

visualization of any custom cross section planes
See also:
The SlicePlaneControllerUI class provides an easy-to-use user interface for the purpose of configuring this renderer.

Queries the scene matrix.

This view's scene matrix is defined by:

\[ S = S_0 \cdot M_0 \cdot M_R \]

with

\[ M_R = \begin{pmatrix} 1 & 0 & 0 & -0.5 \\ 0 & 1 & 0 & -0.5 \\ 0 & 0 & 1 & -0.5 \\ 0 & 0 & 0 & 0 \\ \end{pmatrix} \]

Specifies slice plane.

The plane is defined by \( \vec x : \langle \vec x, \frac{\vec n}{\| \vec n \|} \rangle = d\), where \( \vec n \) is normal and \( d \) is distance.

Precondition:
Carna::base::isEqual( normal.norm(), 0. ) == false
Parameters:
