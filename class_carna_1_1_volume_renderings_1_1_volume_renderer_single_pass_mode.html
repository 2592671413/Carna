Detailed Description

Abstract implementation of VolumeRendererMode, suitable for all further implementations which do require only a single rendering pass for the volume evaluation with some given shader.

Limitations

In contrast to renderColor, for renderDepth only a null-implementation is provided.

Usage

When using this class, you must provide an implementation for configureShader.
