Detailed Description

Displays the orthogonal cross-sections \( \langle e_i,\ x \rangle = d_i, i = \{ 1, 2, 3 \} \) of the volume data set, \( d_i \) being variable.

The image below shows an exemplary rendering result of this class.

Exemplary rendering result of the DefaultAxialPlaneRenderer class
See also:
This renderer is part of the DefaultAxialPlaneVisualization module.

The rendered perspective is fully determined by the utilized camera. This means in particular that the camera determines which planes actually are visible.

The intersection point \( ( d_1, d_2, d_3 )^\mathrm T \) defines the position of the planes. It must be set using volume space coordinates. The intersection point is the one point that is common to all three cross-sectional planes.

Also refer to the interface documentation for details on how to customize this renderer's behavior.

This view's scene matrix is defined by:

\[ S = S_0 \cdot M_0 \cdot M_R \]

with

\[ M_R = \begin{pmatrix} 1 & 0 & 0 & -0.5 \\ 0 & 1 & 0 & -0.5 \\ 0 & 0 & 1 & -0.5 \\ 0 & 0 & 0 & 0 \\ \end{pmatrix} \]
