Detailed Description

Represents the visualization environment for offscreen rendering.

See also:
Refer to OffscreenDisplay for details on offscreen rendering.

The environment can operate both, providing an exclusive OpenGL context or sharing the context with any other VisualizationEnvironment, such as that of a regular qt::Display.
